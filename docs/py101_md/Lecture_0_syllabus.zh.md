## 课程描述
- **官方课程网页：** 人工智能程序设计 （Python）
本课程专为经济学院及相关学院的二年级本科生设计。作为课程体系的核心组成部分，旨在培养学生掌握必要的Python编程技能，理解Python解释器，并能够运用主流的机器学习（ML）和人工智能（AI）模型分析经济学中的大规模数据集。

### 学习目标
成功完成本课程后，学生将能够：
- 有效管理Python环境
- 使用Python开发环境
- 理解并应用Python运算符和关键字
- 使用各种数据结构和容器
- 开发和使用函数
- 掌握面向对象编程（OOP）的基础知识
- 使用 `numpy` 和 `pandas` 进行数据分析
- 执行数据整理和创建可视化图表
- 实现入门级机器学习算法

### 主要内容
- Python环境管理与开发工具
- Python基础：运算符、关键字、数据结构、容器
- Python中的函数与面向对象编程（OOP）
- 使用 `numpy` 和 `pandas` 进行数据分析
- 数据整理与可视化技术
- **机器学习入门算法：**
    - 线性回归
    - 逻辑回归
    - AdaBoost
- **进阶/其他主题：**
    - 性能优化：加速Python
    - 模型评估：损失函数与预测指标
    - 交叉验证技术
    - 优化算法：随机梯度下降（SGD）、贝叶斯优化
    - 神经网络：反向传播
